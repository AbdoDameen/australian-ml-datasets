🇦🇺 Australian Machine Learning Datasets

This repository provides a curated collection of high-quality Australian datasets,
cleaned and structured for machine learning applications.

The goal is to make publicly available Australian data more accessible and usable
for data scientists, researchers, and developers.

🔍 What’s included:
- Cleaned and preprocessed datasets
- Feature-engineered versions
- Time-series and tabular data
- Clear metadata and documentation
- Example ML use cases

📊 Domains covered:
- Demographics (ABS Census)
- Weather (Bureau of Meteorology)
- Transport (NSW Opal & traffic data)
- Housing & real estate
- Health & crime statistics
- Economic indicators (RBA)

🚀 Use cases:
- Regression & classification
- Time-series forecasting
- Clustering & anomaly detection
- Geospatial analysis

📌 Data Sources:
All datasets are sourced from official Australian government portals and
publicly उपलब्ध platforms such as data.gov.au.

⚠️ Disclaimer:
Datasets are redistributed under their respective licenses. Users must verify
usage rights before commercial use.
